#  Wen Secure

AI-powered Ethereum smart contract security audit tool that helps identify vulnerabilities, gas optimizations, and code quality issues.

## Features

- **Automatic Contract Detection**: Fetches verified contract source code from Etherscan
- **AI Security Analysis**: Uses Claude/GPT to analyze contracts for security vulnerabilities
- **Comprehensive Reports**: Detailed audit reports with severity classifications
- **Export Options**: Download reports as PDF or Markdown
- **Real-time Progress**: Visual progress indicators during audit process
- **Responsive Design**: Mobile-friendly interface with modern UI
